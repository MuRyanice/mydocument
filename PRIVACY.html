隐私政策

最后更新日期：2024.3

1. 引言

欢迎使用 Deep Research 浏览器扩展（以下简称"扩展"）。本隐私政策旨在说明我们如何收集、使用、存储和保护您的信息。我们深知隐私的重要性，并致力于保护您的个人数据。

2. 信息收集与使用

2.1 API密钥

  • 我们收集并存储您提供的 ChatGPT 和 DeepSeek API 密钥。
  • 此密钥仅存储在您的本地浏览器中，用于访问相应的 AI 服务。
  • 我们不会将您的 API 密钥传输或存储到我们的服务器上。

2.2 选定文本

  • 当您使用扩展功能时，我们会处理您在网页上选择的文本。
  • 此文本仅用于生成 AI 响应，不会被永久存储或用于其他目的。

2.3 AI响应

  • 通过 ChatGPT 或 DeepSeek API 生成的 AI 响应仅在您的浏览器中显示。
  • 我们不存储或进一步处理这些响应。

3. 数据存储

所有数据（包括 API 密钥）仅存储在您的本地浏览器中。我们不在任何远程服务器上存储您的个人数据。

4. 数据共享

我们不与任何第三方共享您的个人数据。您选择的文本仅用于通过相应的 AI API 生成响应。

5. 安全措施

我们采取适当的技术措施来保护您的信息。然而，请注意，任何互联网传输都不能保证100%的安全性。

6. 用户权利

您可以随时通过删除扩展来移除所有本地存储的数据。您也可以在扩展设置中更改或删除您的 API 密钥。

7. 政策更新

我们可能会不时更新本隐私政策。任何重大变更都会通过扩展更新通知您。
